# Primer ejercicio - "Hola Mundo" (IA & Big Data)

**Autor:** Alfredo López Nogueroles  

## Resumen
Este repositorio contiene mi primer ejercicio práctico para el curso: crear una cuenta en GitHub, inicializar un repositorio y subir un script en Python que muestre el mensaje "Hola mundo!". El objetivo principal es familiarizarme con GitHub.

## Objetivos de la actividad
1. Registrar una cuenta en GitHub
2. Crear un repositorio público y subir un `README.md` y un `holamundo.py`.
3. Conectar con comunidades relacionadas en LinkedIn en mi caso porque no tengo Twitter.

Desde la carpeta del repositorio, en una terminal:
